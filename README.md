# 📱 基于知识蒸馏的轻量化短文本安全分类研究

本项目致力于解决在资源受限设备（如移动端、边缘设备）上进行短文本安全过滤的难题。通过引入**知识蒸馏（Knowledge Distillation）**技术，我们将大型预训练模型（BERT）的“知识”高效迁移至轻量化模型（TinyBERT），在保证高精度的同时，实现了极低的计算开销与高速推理。

> **🚀 核心亮点**：
> - **极致轻量化**：参数量仅 **4.4M**，仅为 BERT-base 的 1/25。
> - **性能逼近 SOTA**：F1-Score 达到 **0.9556**，无限接近 BERT-base。
> - **推理极速**：推理延迟低至 **2.86ms**，相比 BERT-base 提升 **1.7 倍**速度。

---

## 📊 实验结果概览

### 1. 传统机器学习模型表现
我们首先对经典的统计学习模型进行了基准测试。结果显示，尽管这些模型速度快，但在处理复杂的语义安全过滤任务时，召回率（Recall）存在明显短板。

| 模型 | Accuracy | Precision | Recall | F1-Score |
| :--- | :--- | :--- | :--- | :--- |
| **Naive Bayes (NB)** | 0.9748 | 0.9812 | 0.8462 | 0.9087 |
| **SVM (Linear)** | 0.9821 | 0.9836 | 0.8824 | 0.9402 |
| **Logistic Regression (LR)**| 0.9785 | 0.9854 | 0.8647 | 0.9211 |
| **Random Forest (RF)** | 0.9785 | 0.9957 | 0.7853 | 0.9412 |

### 2. 深度学习与预训练模型对比
我们将深度学习架构（Text-CNN）与预训练模型（BERT-base, TinyBERT）进行了对比。数据清晰地展示了预训练模型在该任务上的统治力，以及 TinyBERT 的优越性价比。

| 模型 | Accuracy | Precision | Recall | **F1-Score** | **Latency (ms)** |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **BERT-base** | 0.9910 | 0.9868 | 0.9494 | **0.9677** | 4.83 |

## 📈 性能与延迟综合分析

为了直观展示模型在“性能”与“延迟”之间的权衡，我们绘制了综合对比图。横坐标为不同模型，左侧纵坐标为 F1 分数（越高越好），右侧纵坐标为推理延迟（越低越好）。

### 🔍 关键发现：
1.  **BERT-base (蓝色柱状图)**：拥有最高的 F1 分数（接近 1.0），但延迟最高（约 5ms），不适合实时性要求极高的场景。
2.  **Text-CNN (深蓝色柱状图)**：速度极快（<1ms），但牺牲了大量精度，F1 分数最低。
3.  **TinyBERT (中间蓝色柱状图)**：**完美平衡点**。它将延迟降低到 BERT 的 1/1.7，同时保留了绝大部分的精度优势。

## 🎓 结论

本研究表明，在短文本安全过滤场景中：
1.  **传统模型**（如 SVM, LR）由于缺乏深层语义理解，容易出现漏检（低召回率）。
2.  **纯深度学习模型**（Text-CNN）速度快但精度受限。
3.  **TinyBERT** 通过知识蒸馏技术，成功地将大模型的鲁棒性迁移到小模型中。

**最终推荐**：对于需要兼顾**高精度**与**高并发**的移动端/边缘端应用，**TinyBERT** 是当前的最佳实践方案。
